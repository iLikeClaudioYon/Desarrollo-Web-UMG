¿Qué es un sistema operativo?


Un Sistema Operativo es un programa o un conjunto de programas que hacen efectivo al hardware de un sistema. Estos programas trabajan juntos para lograr ciertos objetivos. Un sistema operativo se encarga de gestionar ciertos recursos como entrada/salida, tratamiento de la información y almacenamiento masivo.

Objetivos de los S.O.

-Hacer posible el uso eficiente de los recursos del sistema.
-Simplificar la operación de trabajo con el hardware.

Funciones de los S. O.

-Asignación de tiempo de CPU
-Control de recursos
-Control de entrada/salida
-Control de errores y protección
-Interfaz con el usuario
-Facultades contables
